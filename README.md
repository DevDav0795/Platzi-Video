## JavaScript Profesional

**Pasos a Seguir**

1. Iniciar un proyecto con  
    `npm init -y` 
2. Instalar dependencia de live server:  
    `npm i -D live-server`
3. Agregar comando a `scripts` en `package.json`
    `"start": "live-server"` 
    3. iniciar proyecto
        `npm start`
